❓ Где получить токен бота?

✅ Ответ:
   1. Откройте Telegram и найдите @BotFather
   2. Напишите /newbot
   3. Следуйте инструкциям
   4. @BotFather отправит вам токен

   Вот прямая ссылка: https://t.me/botfather


❓ Как узнать свой User ID?

✅ Ответ:
   1. Найдите бота @userinfobot в Telegram
   2. Напишите ему любое сообщение
   3. Бот отправит вашу информацию с User ID

   Прямая ссылка: https://t.me/userinfobot


❓ Как изменить формат даты?

✅ Ответ:
   Текущий формат: ДД.МММ.ГГГГ (16.02.2026)

   Для изменения отредактируйте в handlers.py строки:
   datetime.strptime(date_str, "%d.%m.%Y")

   Доступные форматы:
   - "%d.%m.%Y" → 16.02.2026
   - "%d/%m/%Y" → 16/02/2026
   - "%Y-%m-%d" → 2026-02-16


❓ Как добавить нескольких администраторов?

✅ Ответ:
   Отредактируйте config.py:

   # Вариант 1: Список админов
   ADMIN_IDS = [123456789, 987654321]

   Затем в handlers.py измените проверку:
   if message.from_user.id in ADMIN_IDS:

   Вместо:
   if message.from_user.id == ADMIN_ID:


❓ Как запустить бота на серверу?

✅ Ответ:
   Есть несколько способов:

   1. VPS (виртуальный сервер):
      - Арендуйте VPS (например, на Timeweb, VDSina)
      - Установите Python 3.9+
      - Загрузите файлы бота
      - Установите зависимости: pip install -r requirements.txt
      - Запустите: python main.py
      - Используйте screen или systemd для фонового запуска

   2. Heroku (бесплатное хостинг - может быть неактуально):
      - Создайте аккаунт на heroku.com
      - Создайте файл Procfile с содержимым:
        web: python main.py
      - Загрузите через git

   3. Docker:
      - Создайте Dockerfile
      - Соберите образ
      - Запустите контейнер

   Рекомендуем использование systemd для автозагрузки:
   
   Создайте файл /etc/systemd/system/homework-bot.service:
   ─────────────────────────────────────
   [Unit]
   Description=Homework Bot
   After=network.target

   [Service]
   Type=simple
   User=your_user
   WorkingDirectory=/path/to/bot
   ExecStart=/usr/bin/python3 /path/to/bot/main.py
   Restart=always

   [Install]
   WantedBy=multi-user.target
   ─────────────────────────────────────

   Затем:
   sudo systemctl enable homework-bot
   sudo systemctl start homework-bot

✅ Ответ:
   1. Остановите бота
   2. Удалите файл homework.db из папки проекта
   3. Запустите бота заново (база пересоздастся)


❓ Как добавить больше команд?

✅ Ответ:
   В handlers.py добавьте новый обработчик:

   ─────────────────────────────────────
   @router.message(Command("your_command"))
   async def your_handler(message: Message):
       await message.answer("Текст ответа")
   ─────────────────────────────────────

   Например, команда /stats:

   ─────────────────────────────────────
   @router.message(Command("stats"))
   async def stats(message: Message):
       all_hw = db.get_all_homework()
       await message.answer(f"📊 Всего заданий: {len(all_hw)}")
   ─────────────────────────────────────


❓ Как изменить сообщения бота?

✅ Ответ:
   Все сообщения находятся в handlers.py. Например:

   Найдите строку:
   await message.answer("👋 Привет!")

   И измените текст:
   await message.answer("🎉 Добро пожаловать!")

       await message.answer(welcome_text, reply_markup=keyboard)

💡 Совет 3: Поиск по ключевым словам

Добавьте в database.py:

   def search_homework(self, keyword: str):
       conn = sqlite3.connect(self.db_path)
       cursor = conn.cursor()
       cursor.execute(
           "SELECT date, text FROM homework WHERE text LIKE ?",
           (f"%{keyword}%",)
       )
       results = cursor.fetchall()
       conn.close()
       return results


💡 Совет 4: Экспорт всех заданий в Excel

Установите openpyxl:
   pip install openpyxl

Добавьте в handlers.py:

   @router.message(Command("export"))
   async def export_homework(message: Message):
       if message.from_user.id != ADMIN_ID:
           return
       
       from openpyxl import Workbook
       
       wb = Workbook()
       ws = wb.active
       ws.append(["Дата", "Задание"])
       
       for date, text, _ in db.get_all_homework():
           ws.append([date, text])
       
       wb.save("homework.xlsx")
       await message.answer_document(
           open("homework.xlsx", "rb")
       )
