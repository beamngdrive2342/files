📋 СОДЕРЖАНИЕ КАЖДОГО ФАЙЛА

📄 main.py:
   • Импорты и логирование
   • Функция set_commands() - установка команд
   • Функция on_startup() - действия при запуске
   • Функция on_shutdown() - действия при остановке
   • Функция main() - главный цикл
   • Обработка исключений

📄 config.py:
   • BOT_TOKEN - токен бота
   • ADMIN_ID - User ID администратора
   • DATABASE_PATH - путь к БД
   • DEBUG_MODE - режим отладки

📄 database.py:
   • Класс Database
   • init_db() - инициализация БД
   • add_homework() - добавление
   • get_homework() - получение по дате
   • update_homework() - обновление
   • delete_homework() - удаление
   • get_all_homework() - получение всех
   • homework_exists() - проверка существования

📄 handlers.py:
   • FSM стейты (AddHomeworkStates, EditHomeworkStates, DeleteHomeworkStates)
   • Команды: /start, /help
   • Админ панель с обработчиками
   • Добавление задания (3 этапа)
   • Редактирование задания
   • Удаление задания
   • Просмотр всех заданий
   • Выбор даты и показ задания
