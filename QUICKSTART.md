🎯 ШАГ 1️⃣  ПОЛУЧИТЕ ТОКЕН (2 МИНУТЫ)

1. Откройте Telegram → найдите @BotFather
2. Напишите /newbot
3. Назовите бота (например: "HomeWork Bot")
4. Дайте username (например: my_homework_bot_2025)
5. Скопируйте токен вида: 123456789:ABCDEfgh...


🎯 ШАГ 2️⃣  УЗНАЙТЕ СВОЙ ID (1 МИНУТА)

1. В Telegram найдите @userinfobot
2. Напишите ему любое сообщение
3. Скопируйте User ID (число типа 987654321)


🎯 ШАГ 3️⃣  УСТАНОВКА (2 МИНУТЫ)

1. Создайте папку: homework_bot
2. Скопируйте туда все файлы (.py, .txt)
3. Откройте config.py и вставьте:
   ├─ BOT_TOKEN = "ВАШ_ТОКЕН"
   └─ ADMIN_ID = ВАШ_ID

4. Откройте командную строку в этой папке
5. Напишите: pip install -r requirements.txt
6. Дождитесь установки


🎯 ШАГ 4️⃣  ЗАПУСК (1 МИНУТА)

1. В командной строке напишите: python main.py
2. Если все OK, увидите:
   ✅ Команды установлены
   📡 Запуск polling...

3. Откройте Telegram, найдите своего бота по username
4. Нажмите /start

✨ ВСЁ РАБОТАЕТ! ✨

❓ ЕСЛИ ЧТО-ТО НЕ РАБОТАЕТ

❌ "Invalid token provided"
→ Проверьте токен в config.py (скопируйте без пробелов)

❌ "No module named aiogram"
→ Установите: pip install -r requirements.txt

❌ Бот не отвечает
→ Убедитесь что окно командной строки открыто
→ Перезагрузите: Ctrl+C и python main.py

❌ "только администратор"
→ Проверьте ADMIN_ID в config.py (должен быть ваш ID)
