📚 ИНСТРУКЦИЯ ПО УСТАНОВКЕ И ЗАПУСКУ БОТА ДЛЯ ДОМАШНИХ ЗАДАНИЙ

🔧 ПРЕДВАРИТЕЛЬНЫЕ ТРЕБОВАНИЯ:

1. Python 3.9 или выше (скачать с python.org)
   - При установке ОБЯЗАТЕЛЬНО отметьте "Add Python to PATH"

2. Git (опционально, для клонирования репозитория)

3. Telegram (приложение или веб-версия)


═══════════════════════════════════════════════════════════════════════════════

📋 ШАГ 1: ПОЛУЧЕНИЕ ТОКЕНА БОТА

1. Откройте Telegram и найдите бота @BotFather
   (или перейдите по ссылке: https://t.me/botfather)

2. Напишите команду: /start

3. Затем напишите: /newbot

4. @BotFather попросит вас ввести имя бота
   Пример: "MyHomeworkBot"

5. @BotFather попросит ввести username бота
   Пример: "my_homework_bot_2025"
   (Должен быть уникальным и заканчиваться на "_bot")

6. @BotFather отправит вам ТОКЕН бота
   Выглядит как: 123456789:ABCDEfghijklmnopqrSTUVwxyz

⚠️  СОХРАНИТЕ ЭТОТ ТОКЕН! Он вам понадобится.


═══════════════════════════════════════════════════════════════════════════════

👤 ШАГ 2: ПОЛУЧЕНИЕ СВОЕГО USER ID

1. Найдите бота @userinfobot в Telegram
   (или перейдите по ссылке: https://t.me/userinfobot)

2. Напишите ему любое сообщение

3. Бот отправит вам информацию с вашим User ID
   Выглядит как число (например: 987654321)

⚠️  ЗАПОМНИТЕ ИЛИ СОХРАНИТЕ ЭТОТ ID! Это будет ADMIN_ID.


═══════════════════════════════════════════════════════════════════════════════

💾 ШАГ 3: ЗАГРУЗКА И ПОДГОТОВКА ПРОЕКТА

Вариант A - Скачивание ZIP архива:

1. Скачайте все файлы проекта:
   - config.py
   - database.py
   - handlers.py
   - main.py
   - requirements.txt
   - README.md (этот файл)

2. Создайте папку для проекта (например: homework_bot)

3. Поместите все файлы в эту папку

Вариант B - Через командную строку:

1. Откройте командную строку (Windows: cmd.exe, Mac/Linux: terminal)

2. Перейдите в нужную папку:
   cd путь/к/папке

3. Клонируйте репозиторий (если используется Git):
   git clone [ссылка_на_репозиторий]


═══════════════════════════════════════════════════════════════════════════════

⚙️  ШАГ 4: НАСТРОЙКА КОНФИГУРАЦИИ

1. Откройте файл config.py в текстовом редакторе (например, Notepad++)

2. Найдите строку:
   BOT_TOKEN = "YOUR_BOT_TOKEN_HERE"

3. Замените на свой токен:
   BOT_TOKEN = "123456789:ABCDEfghijklmnopqrSTUVwxyz"

4. Найдите строку:
   ADMIN_ID = 123456789

5. Замените на свой User ID:
   ADMIN_ID = 987654321

6. Сохраните файл (Ctrl+S)

Пример готовой конфигурации:
─────────────────────────────────────
BOT_TOKEN = "987654321:ABCdefGHIjklmnoPQRstuvWXYZ123"
ADMIN_ID = 123456789
DATABASE_PATH = "homework.db"
DEBUG_MODE = True
─────────────────────────────────────


═══════════════════════════════════════════════════════════════════════════════

📦 ШАГ 5: УСТАНОВКА ЗАВИСИМОСТЕЙ

1. Откройте командную строку в папке проекта

2. На Windows:
   pip install -r requirements.txt

   На Mac/Linux:
   pip3 install -r requirements.txt

3. Дождитесь завершения установки (обычно занимает 1-2 минуты)

Если возникает ошибка, попробуйте:
   pip install --upgrade pip
   pip install aiogram==3.3.0
   pip install python-dotenv==1.0.0


═══════════════════════════════════════════════════════════════════════════════

🚀 ШАГ 6: ЗАПУСК БОТА

1. Откройте командную строку в папке проекта

2. На Windows:
   python main.py

   На Mac/Linux:
   python3 main.py

3. Если всё работает, вы увидите:
   ✅ Команды установлены
   📡 Запуск polling (слушание сообщений)...

⚠️  НЕ ЗАКРЫВАЙТЕ КОМАНДНУЮ СТРОКУ! Бот работает пока она открыта.

🐛 РЕШЕНИЕ ПРОБЛЕМ

❌ Проблема: "ModuleNotFoundError: No module named 'aiogram'"

✅ Решение:
   - Убедитесь, что установили зависимости: pip install -r requirements.txt
   - На Mac/Linux используйте: pip3 install -r requirements.txt


❌ Проблема: "Invalid token provided"

✅ Решение:
   - Проверьте, что правильно скопировали токен в config.py
   - Убедитесь, что нет пробелов в начале или конце токена
   - Получите новый токен у @BotFather


❌ Проблема: Бот не отвечает

✅ Решение:
   - Убедитесь, что командная строка с ботом ещё открыта
   - Перезагрузите бота (Ctrl+C и заново python main.py)
   - Проверьте интернет-соединение
   - Убедитесь, что у вас есть доступ к Telegram


❌ Проблема: Фото не отправляются

✅ Решение:
   - Убедитесь, что размер фото не больше 20 МБ
   - Отправляйте фото одно за одним
   - Попробуйте повторить процесс добавления

📝 ПРИМЕЧАНИЯ

• Дата должна быть в формате ДД.МММ.ГГГГ (например: 16.02.2026)
  - ДД = день (01-31)
  - МММ = месяц (01-12)
  - ГГГГ = год (2024, 2025 и т.д.)

• Можно добавлять несколько фотографий к одному заданию

• База данных автоматически сохраняется в файл homework.db

• Бот использует SQLite - не требует установки дополнительных серверов БД

• Все заданий хранятся локально на вашем компьютере
